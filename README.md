# CampusOps

CampusOps is an academic management platform designed to manage:

- Course scheduling
- Student absences
- Academic progress
- Payment tracking
- Email notifications
- Telegram bot commands
- OpenClaw automation workflows

## Architecture

Frontend: React  
Backend: Node.js + Express  
Database: PostgreSQL  
Automation: OpenClaw  
Messaging: Telegram Bot  
Email: IMAP / SMTP  

## Project Structure

backend/ → API and business logic  
frontend/ → Web dashboard  
docs/ → diagrams and documentation  
workflows/ → OpenClaw automation  
presentation/ → demo materials
